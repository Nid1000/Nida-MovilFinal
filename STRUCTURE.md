# Estructura del proyecto

Este repositorio contiene la aplicación móvil Flutter y su backend API Laravel.

```text
APPNI-main/
  android/
  assets/
  backend/
  ios/
  lib/
  test/
  analysis_options.yaml
  pubspec.lock
  pubspec.yaml
  README.md
```

No incluye una aplicación web ni plataformas de escritorio.

`.dart_tool/` y `build/` son carpetas generadas y no deben subirse.
